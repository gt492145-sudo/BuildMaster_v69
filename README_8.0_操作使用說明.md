# Construction Master V8.0 操作使用說明

本文件提供 `BuildMaster_v69`（V8.0 模組化版）之現場操作指引，適用於估算、放樣、量測與 QA 稽核流程。

## 1. 版本定位

- 目前版本為 V8.0 模組化架構（`modules/v80-*.js`）。
- `index.html` 已轉為入口頁，核心功能分散於各專責模組。
- 放樣與量測功能已整合「測量加強版」能力（交會法、導線閉合、多段坡度、CSV/儀器格式匯出等）。

## 2. 快速開始（第一次使用）

1. 開啟 `index.html`。
2. 選擇地區價目來源（影響材料單價）。
3. 上傳圖紙或模型檔（依作業情境）。
4. 設定放樣精度（E/N/H）與座標系統（TWD97/TWD67/雙顯示）。
5. 執行量測或放樣流程後，做 QA 檢核。
6. 匯出結果（放樣點、QA 報告、施工包或專案檔）。

## 3. 主要工作模式

- **估算模式**：材料、工程量、進階估價報表。
- **放樣模式**：模型抽點、分群、高精度修正、放樣 QA、施工包輸出。
- **量圖模式**：圖紙定比例、距離量測、嚴格模式傾斜管控、量圖 QA 報告。

## 4. 放樣作業建議流程（現場標準）

1. 匯入 BIM 模型。
2. 產生放樣點（柱、牆、梁等）。
3. 執行分群與品質強化（必要時重測）。
4. 執行放樣 QA。
5. 檢查 QA 指標（重複、缺漏、越界、穩定度）。
6. 通過後匯出施工包（高/中/禁分級）。

## 5. 企業 QA（嚴格模式）

V8.0 已提供企業級 QA 審核流程：

- 可切換 QA 設定檔：`企業級（嚴格）` / `標準版`
- 可一鍵執行：`企業 QA 審核`
- 可匯出：`企業 QA 報告`
- 若企業審核未通過，施工包匯出會被阻擋

建議日常使用 `企業級（嚴格）`，僅在測試或教育情境使用標準版。

## 6. 測量加強版功能清單

- E/N/H 小數位精度設定（3/4/5）
- TWD97 / TWD67 / 雙座標顯示
- 反算（E,N -> X,Y）
- 兩點距離與方位角
- 極座標放樣與加入點表
- 交會法（距離交會、方位交會）
- 導線閉合差（含 Bowditch 建議）
- 多段坡度計算
- 面積單位換算（m2/坪/公頃/甲）
- CSV 匯入（含錯誤行提示）
- 儀器格式匯出（Leica/Trimble/Topcon）
- 專案存檔/載入（localStorage/JSON）
- 快捷鍵（Enter/Esc/Ctrl+Z）
- 格網與比例尺視覺輔助

## 7. 常見排錯

- **看不到放樣點**：先確認模型已載入，且類型勾選正確。
- **QA 分數偏低**：先做高精度修正、分群 QA，再重跑放樣 QA。
- **施工包無法匯出**：檢查天氣聯動門檻與企業 QA 阻擋原因。
- **量測誤差大**：重新定比例，並確認裝置傾斜角未超出嚴格模式門檻。
- **更新後畫面不一致**：重新整理並清除舊快取（Service Worker 已版本化）。

## 8. 發佈與快取注意事項

- 每次有模組變更，需同步更新 `service-worker.js` 的 `CACHE_VERSION`。
- 新增模組檔案必須加入 `CORE_ASSETS`，避免離線或舊版快取漏載。

## 9. 檔案結構建議（維運）

- 核心入口：`index.html`
- 模組目錄：`modules/v80-*.js`
- 快取控制：`service-worker.js`
- 兼容導向頁：`stake.html`
